    Rift Guide

    Core Loop

    Every round shows an abstract puzzle in the arena and four possible answers below it. The right panel names the active chamber and the exact rule. Pick the answer that satisfies that rule before the timer bar empties.

    Correct answers raise score and streak. Every third hit increases level. Misses lower level, so the app keeps drifting toward the edge of what you can still hold together.

    Controls

    AnswerClick or tap an answer. Keyboard: 1-4 or A-D.
    PauseUse the Pause button or press P.
    SoundStart Rift uses tones. Silent Start keeps the same game muted.
    ResetReset returns to the launch state and clears the current run.

    Chambers

    Echo LatticeN-back. Choose the object from N trials ago. The prompt names which features matter.
    Contradiction ForgeStroop. If the prompt says INK, obey color. If it says WORD, obey the written color name.
    Mirror TorsionMental rotation. Choose the same object after rotation. Reflections are traps.
    Vanishing ArrayChange detection. Study the array, survive the mask, then choose the changed cell.
    Operator DreamAnalogy. Infer the hidden A to B transform and apply it to C.
    Tensor CrucibleFeature synthesis. Build one answer object from channels split across A, B, and C.

    Customization

    Use Tune to reshape the machine. Lower time scale, higher bind load, more noise, deeper memory, more objects, and harsher adaptation all raise strain. Chamber toggles let you build a focused nightmare or a full mixed gauntlet.

    Reading Objects

    Objects can differ by glyph text, outline shape, color, small orbit dots, ring count, rotation, and mirror direction. The hard part is not seeing them one by one; it is binding several at once while the current rule changes.

    Training transfer is debated; treat this as a hard cognitive game.
